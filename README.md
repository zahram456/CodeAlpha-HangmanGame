# CodeAlpha Hangman Game

A simple terminal-based Hangman game written in Python.

## Features
- Randomly selects a word from a predefined list
- Tracks guessed letters
- Allows up to 6 wrong attempts
- Displays win/lose result at the end

## Requirements
- Python 3.x

## Run
```bash
python hangman.py
```

## How to Play
- Enter one letter per turn.
- If the letter is in the word, all matching positions are revealed.
- If the letter is not in the word, one attempt is used.
- You win by guessing all letters before using 6 wrong attempts.

## Project Structure
- `hangman.py`: Main game script
